# Full Adder Layout Design (Sky130 – Magic VLSI)

## Overview
This project presents the transistor-level CMOS layout of a Full Adder designed using the Sky130 PDK in Magic VLSI.

## Design Details
- Implemented Full Adder functionality using combinational CMOS logic
- Designed at transistor level including Sum and Carry outputs
- Layout created following Sky130 design rules

## Tools & Technology
- Magic VLSI
- Sky130 PDK

## Verification
- Performed Design Rule Check (DRC) to ensure layout correctness
- Layout follows basic design constraints and rules
